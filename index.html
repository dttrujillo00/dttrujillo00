Habilidades

HTML5

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web.

CSS

CSS es el lenguaje de estilos utilizado para describir la presentación de documentos HTML.CSS describe como debe ser renderizado el elemento estructurado en la pantalla

JavaScript

JavaScript es un lenguaje de programación diseñado en un principio para añadir interactividad a las páginas webs y crear aplicaciones web. Junto al HTML y a CSS, es una de las tres tecnologías esenciales en la producción mundial webs

React

React es una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página. Es mantenido por Facebook y la comunidad de software libre

Sass

Sass es un preprocesador CSS, es decir, que coge el código programado en otro lenguaje (Sass, SCSS) y lo transforma en código CSS.Proporciona estructuras avanzadas propias de los lenguajes de programación, como variables, listas, funciones y estructuras de control.

PWA

Una aplicación web progresiva o PWA, es un sitio web que se ve y se comporta como si fuera una aplicación móvil. Entre sus ventajas está que aprovecha las funciones nativas de los dispositivos móviles y brinda una mejor experiencia de usuario
